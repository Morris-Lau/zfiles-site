Summary

ZFiles does not collect, sell, share, or track personal data. The app does not include advertising, analytics, or tracking SDKs.

Data Stored on Your Device

ZFiles stores app data locally on your device so that the app can work as a file browser. This may include:

  • server connection settings such as server name, host, port, protocol type, username, share name, and initial path;
  • bookmarks, tags, browser preferences, and cache metadata;
  • temporary files, thumbnails, and preview cache used to display files you choose to browse.

Passwords are stored in the iOS Keychain. Trusted SFTP host fingerprints are also stored locally for connection security.

Network Connections

ZFiles connects only to file servers that you configure or choose to access. These can include SMB, SFTP, and WebDAV servers. When you connect to those servers, the information needed for that connection is sent to the server you selected.

ZFiles uses local network access to discover and connect to file servers on your local network. ZFiles does not operate a backend service for your files.

Files and User Content

ZFiles can display, preview, upload, download, copy, move, rename, and delete files when you request those actions. File contents are processed on your device or transferred between your device and the file server you selected.

The developer does not receive your files, server addresses, credentials, bookmarks, tags, or browsing history.

Third-party Code

ZFiles uses third-party open source packages for file transfer, networking, cryptography, and archive support. These packages run inside the app and are used to provide app functionality. See Third-party Notices.

隐私政策摘要

ZFiles 不收集、出售、共享或追踪个人数据。应用不包含广告、分析 SDK 或追踪 SDK。

本机保存的数据

ZFiles 会在你的设备本机保存应用正常运行所需的数据，包括服务器连接设置、书签、标签、浏览偏好、缓存元数据、临时文件、缩略图和预览缓存。

服务器密码保存在 iOS 钥匙串中。受信任的 SFTP 主机指纹也会保存在本机，用于连接安全校验。

网络连接

ZFiles 只会连接你自己配置或选择访问的文件服务器，包括 SMB、SFTP 和 WebDAV 服务器。连接这些服务器时，完成连接所需的信息会发送给你选择的服务器。

ZFiles 使用本地网络权限来发现和连接同一局域网中的文件服务器。ZFiles 不为你的文件提供开发者运营的后端服务。

文件与用户内容

ZFiles 可以根据你的操作浏览、预览、上传、下载、复制、移动、重命名和删除文件。文件内容只会在你的设备上处理，或在你的设备与你选择的文件服务器之间传输。

开发者不会收到你的文件、服务器地址、凭据、书签、标签或浏览历史。
